# Calculadora Simples em C++

Este é um projeto simples desenvolvido em C++ com a funcionalidade/tentativa de calcular

O programa possibilita:

- Somar numeros
- subtrair numeros
- multiplicar numeros
- dividir numeros



## Funcionalidades

- Soma ('+')
- Subtração ('-')
- Multiplicação ('*')
- Divisão ('/')



## Como executar 

Compile o arquivo com o comando:
*```bash
g++ calculadora.cpp -o calculadora*
